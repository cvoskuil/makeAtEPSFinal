3D Printer

Description

The "MakerBot Replicator 2" is one of the most high tech 3D printers at EPS. It prints out 3D objects by spreading a thin plastic string based on the specifications described in a file.

Operation

Step 1: Find an object to print. Thingiverse is a good place to start. You can also make your own object using a CAD program.

Step 2: Use MakerBot Desktop to view your object file and prepare it for printing. When you are ready, export the print to a SD card as a *.X3G file.

Step 3: Ensure that the filament color you wish to use is connected to the Replicator 2.

Step 4: Connect the SD card to the Replicator 2.

Step 5: Select "Build from SD" and select your print from the list.

Step 6: Start the print.
